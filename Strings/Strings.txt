/// Mutable and Immutable string

// Immutable string: String
// - Once created, cannot be modified
// - Any modification creates a new object
// - Example: String s = "Hello";

// Mutable string: StringBuffer, StringBuilder
// - Can be modified (append, insert, delete, etc.) without creating new objects

// StringBuilder : 
// - Mutable sequence of characters
// - Not synchronized (not thread-safe)
// - Faster, suitable for single-threaded applications

// StringBuffer : 
// - Mutable sequence of characters
// - Synchronized (thread-safe, safe to use in multi-threaded apps)
// - Slightly slower than StringBuilder due to synchronization
